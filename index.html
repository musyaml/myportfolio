Tentang Saya

Siapa Saya?

Saya adalah seorang mahasiswa Sistem Informasi di Universitas Mercu Buana. Saya mulai menyukai dunia teknologi sejak saya berumur 13 tahun. Berawal dengan orang tua saya yang membelikan saya sebuah smartphone, tetapi tidak kebanyakan anak lainnya yang langsung memakai smartphone mereka begitu saja. Saya menkonfigurasi smartphone saya terlebih dahulu, dimulai mengubah user experience sampai ke user interface, setelah itu baru saya memakainya. Tujuan saya menkonfigurasinya supaya smartphone bisa bekerja lebih optimal dan maksimal. Sejak saat itu saya mulai mengeksplor lebih dalam mengenai dunia teknologi sampai saat ini.
